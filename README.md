PASOS RASPBERRY:
1. Comprobar IP en ambos códigos (servidor_puente.py & interfaz_sensor.py)
2. Abrir la carpeta PROYECTO_1, ir al apartado "Tools" y seleccionar "Open Current Folder in Terminal"
3. En la terminal escribir "python3 servidor_puente.py", deberá mencionar que la conexión fue exitosa
4. Abrir en Thonny el código interfaz_sensor.py, luego ejecutar
5. Seleccionar frecuencia y apretar "Iniciar Captura" o el botón físico
